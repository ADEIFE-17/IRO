# IRO – Issues Reporting Outlet

## Project Overview

IRO (Issues Reporting Outlet) is a mini web-based reporting system developed as a prototype solution for reporting issues affecting AAUA students and members of the host communities (Akungba, Iwaro, Etioro, Ayegunle and surrounding areas).

The platform enables users to submit complaints, receive a unique Complaint ID and track the progress of submitted reports.

---

## Features

- Report community or student-related issues
- Complaint ID generation
- Complaint tracking
- Anonymous reporting option
- Responsive design
- Local storage support
- Mobile-friendly navigation
- Frequently Asked Questions section

---

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` using any modern web browser.
4. Navigate through the available pages.
5. Submit a report using the Report Issue page.
6. Copy the generated Complaint ID.
7. Use the Track Report page to retrieve the report.

---

## Notes

This project is a front-end prototype developed for academic purposes.

Report information is stored using the browser's Local Storage. No server-side database is included.
